# Match Manager

A GUI-tool built with **Qt 6** and **C++17** for testing the strength of 4 player chess engines.

---

## Table of Contents
- [Usage](#usage)

---

## Usage

You can use this application to run matches between two different engine versions whilst observing their gameplay. Choose total number of games, think time per move and how many games you want to run concurrently at your liking. It is recommended
to set the number of games running concurrently to an integer smaller than the number of physical cores available to your system.
